# Projet-Binaire

## Structure du workspace

- `Niveau Premiere/` : Version élève avec des fonctions (convertir_vers_decimal et decimal_vers_base) à compléter.
- `Projet Base/` : Version complète. C'est la version de référence pour les enseignants et qui sera utilisée pour la première activité faite en classe en début d'année.

Les deux projets sont indépendants et peuvent être utilisés séparément. Le projet de niveau première est conçu pour être utilisé plus tard dans l'année, tandis que le projet de base offre une solution en début d'année pour la conversion de nombres.

Les deux projets sont sciendés avec `index.html` pour l'interface utilisateur et `logic.py` pour la logique de conversion. 

## Execution

- Lancer le projet dans un terminal avec la commande `py -m http.server 8000 --bind 127.0.0.1` à la racine du projet.
- Ouvrir un navigateur et aller à l'adresse `http://http://127.0.0.1:8000/` pour accéder à l'interface de conversion de nombres.
- Suivre les instructions à l'écran pour convertir des nombres entre différentes bases (binaire, octal, décimal, hexadécimal).
