# Projet Base - Version sans Flask

Ce dossier est maintenant en architecture plate: un seul fichier `index.html`.

## Lancer

1. Ouvrir `index.html` dans un navigateur (connexion internet requise pour charger Brython via CDN).
2. Utiliser le convertisseur directement, sans serveur Python.

## Fonctionnalites

- Conversion entre bases 2, 8, 10, 16.
- Affichage des etapes de calcul.
- Mode "Arbre des divisions successives".
